## A basic queueing result

For an M/M/1 queue with arrival rate $\lambda$ and service rate $\mu$, the average number of customers in the system is:

$$
L = \frac{\rho}{1 - \rho}, \quad \text{where } \rho = \frac{\lambda}{\mu}
$$

Inline math works too — utilization $\rho$ must stay below $1$ for the queue to remain stable.
