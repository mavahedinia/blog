این آموزش نصب هگزو روی مک هستش! نصب روی لینوکس‌ها هم خیلی شبیه مک هست!

### پیش نیازها
برای نصب `Hexo`، ابتدا باید `NodeJs` رو نصب کرده باشین؛ چون پایه `Hexo` `NodeJs` هست! بعدش باید برین رابط ترمینال هگزو رو نصب کنین. بعد از این‌که هگزو نصب شد، تازه کار ما شروع میشه! 

### نصب npm
نصب `npm` رو مک حداقل خیلی سادس! (یکی از فرقای نصب رو لینوکس و مک، همین یه تیک‌س!) فقط کافیه به [سایت NodeJs](https://nodejs.org/) برین و آخرین نسخه رو دانلود کنین. یکی از اشتباهات رایج اینه که ممکنه نسخه پایدار (stable) رو دانلود کنین، در حالی که نسخه فعلی (current) جلوتر بدرد می‌خوره!
بعد از دانلود هم فقط باید نصبش کنین دیگه! حالا `NodeJS` نصب شد.

### نصب رابط ترمینال Hexo
حالا که `npm` نصب شد، باید `hexo-cli` رو نصب کنیم، برای اینکارم، فقط کافیه دستور زیر رو تو ترمینال بزنیم:
``` bash
$ sudo npm install -g hexo-cli
```
در ادامه مطلب، ساختن بلاگ با هگزو رو آموزش می‌دیم.
 

## شروع کار

### ساخت پروژه
حالا که `Hexo` نصب شد، نوبت اینه که یه پروژه باهاش بسازیم! اولین کاری که باید بکنیم، اینه که تو ترمینال، دایرکتوری‌مونو عوض کنیم به جایی که می‌خوایم بلاگمونو توش ذخیره کنیم (مثلا من تو آدرس `~/hexo/` رفتم:
``` bash
$ cd ~/hexo
```
حالا باید یه پروژه بسازیم، من برای مثال اسمشو می‌زارم BOZ:
``` bash
$ hexo init BOZ
```
 حالا پوشه BOZ ساخته میشه و یسری فایل توشه، الان کل بلاگ تو این پوشه هست. با دستور `cd BOZ` می‌ریم تو پوشه پروژه.

### فایل ‌`_config.yml`
  حالا می‌رسیم به قسمت تنظیمات `Hexo`. این  فایل تنظیمات اصلی بلاگ رو شامل میشه (مثلا آدرس صفحه اصلی سایت و یا تنظیمات پلاگین‌های `Hexo`) اول از همه باید دایرکتوری و آدرس سایت اصلی رو وارد کنیم:
``` yml
url: https://boz.ir/
root: /
```
بعد باید عنوان و توضیحات و ... سایتو وارد کنیم:
``` yml
title: بز!
subtitle: بز نوشته‌های تست!
description: این یک بلاگ جهت آموزش کار با Hexo می‌باشد
author: Mr.Boz
language: fa
```

این‌ها کانفیگ‌های اصلی بودن، ولی دستتون خیلی برای شخصی سازی بازه! مثلا می‌شه تم و غیره دانلود یا نصب کرد و ... .
پیشنهاد حتما یکمی تو خود [سایت هگزو](https://hexo.io) بگردین، اونجا پر تم و پلاگین هست.

در مطلب بعدی نیز به شیوه نوشتن پست می‌پردازیم.
